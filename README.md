# AstrBot 选择性回复 (Selective Reply)

本插件为 AstrBot 提供了一个智能的对话过滤方案，允许 AI 根据自定义准则自主决定是否回复某条消息。

## 核心特性

- **自主拒绝权**：AI 可以根据注入的价值观准则，判断是否对冒昧、无礼或涉及隐私的问题进行“已读不回”。
- **无声拦截机制**：通过特定暗号触发拦截，对方无法感知到任何回复尝试，主打“高冷”人设。
- **自动准则注入**：无需修改全局人设，插件自动在每次对话中为 AI 提供拦截建议。
- **管理员豁免**：内置管理员保护逻辑，确保核心用户的指令永远被执行。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `selective_reply` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 配置项说明

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `intercept_keyword` | 触发拦截的唯一暗号。 | `我不回复(触发拦截)` |
| `refuse_criteria` | 告诉 AI 什么样的情况该选择不回复。 | `见默认值` |
| `admin_qq` | 管理员 QQ 号（豁免拦截）。 | `1591793025` |
