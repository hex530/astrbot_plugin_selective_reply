# 选择性回复 (Selective Reply)

**作者**: 夕小柠 & 陆渊
**版本**: 1.2.8 (稳定版)

### 核心功能
赋予 AI “拒绝权”，通过智能识别冒昧请求、隐私询问或违反底线的对话，实现自主选择性无视（不回复）。

### 更新日志 (v1.2.8)
- **紧急修复**: 解决了 `filter` 模块与 Python 内置函数冲突导致的加载失败问题。
- **稳定性提升**: 优化了装饰器调用语法，全面兼容 AstrBot 最新框架。
- **逻辑优化**: 注入准则逻辑更加稳健。

### 工作逻辑
1. **准则注入**: 消息到达 LLM 前，插件会自动在上下文中注入“拦截准则”（Prompt），提醒 AI 拥有拒绝回复的权利。
2. **暗号识别**: 如果 AI 判定制回复内容触发了底线，并输出了预设的暗号（默认：`我不回复(触发拦截)`），插件将拦截该消息。
3. **无声拦截**: 触发拦截后，插件会彻底销毁回复内容并停止事件传播。对方不会收到任何消息反馈，实现“冷处理”。

### 配置项说明
- `intercept_keyword`: 触发拦截的唯一暗号。
- `refuse_criteria`: 注入给 AI 的拦截准则（Prompt）。
- `admin_qq`: 管理员 QQ 号。
